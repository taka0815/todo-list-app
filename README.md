# タスク管理ボード

Vue.js 3を使用したシンプルなタスク管理アプリケーションです。カンバン方式でタスクを視覚的に管理できます。

## 機能

- **タスクの作成・削除**: 新しいタスクを追加し、不要になったタスクを削除できます
- **ドラッグ&ドロップ**: タスクをドラッグして別のステータスに移動できます
- **7段階のステータス管理**:
  - 未着手
  - 判断待ち
  - 処理中
  - レビュー待ち
  - テストケース作成中
  - テスト中
  - 終了
- **コメント機能**: 各タスクにコメントを追加・編集・削除できます
- **自動削除**: 完了したタスクは2週間後に自動的に削除されます
- **ローカルストレージ**: データはブラウザのlocalStorageに保存されます

## 使い方

### 起動方法

1. リポジトリをクローンまたはダウンロードします
2. [index.html](index.html)をブラウザで開きます

```bash
# ローカルサーバーを使用する場合（推奨）
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

3. ブラウザで`http://localhost:8000`にアクセスします

### 基本操作

#### タスクの追加
1. 画面上部のフォームにタスク名を入力
2. ステータスを選択（デフォルトは「未着手」）
3. 「タスク追加」ボタンをクリック

#### タスクの移動
- タスクカードをドラッグして別のステータス列にドロップします

#### コメントの追加
1. タスクカードをクリックして詳細パネルを表示
2. コメント入力欄にテキストを入力
3. 「追加」ボタンをクリック

#### タスクの削除
1. タスクカードをクリックして詳細パネルを表示
2. 「タスク削除」ボタンをクリック
3. 確認ダイアログで「OK」を選択

## 技術スタック

- **Vue.js 3**: フロントエンドフレームワーク（CDN版を使用）
- **Vanilla CSS**: スタイリング
- **localStorage API**: データ永続化

## データ構造

### タスクオブジェクト

```javascript
{
  id: string,           // ユニークID
  title: string,        // タスク名
  status: string,       // ステータス
  createdAt: string,    // 作成日時（ISO 8601形式）
  completedAt: string,  // 完了日時（ISO 8601形式、未完了の場合はnull）
  comments: []          // コメント配列
}
```

### コメントオブジェクト

```javascript
{
  id: string,          // ユニークID
  text: string,        // コメント本文
  createdAt: string,   // 作成日時（ISO 8601形式）
  updatedAt: string    // 更新日時（ISO 8601形式、未編集の場合はnull）
}
```

## ブラウザ対応

モダンブラウザに対応しています（Chrome, Firefox, Safari, Edge最新版）。

- ES6+機能を使用しているため、古いブラウザでは動作しない場合があります
- localStorage APIが必要です
